GitHub Integration

Connect to your GitHub repo to publish product changes directly.

ℹ️
How it works: When you click Publish to GitHub, the dashboard pushes your updated products.json to your GitHub repository using the GitHub API. GitHub Pages then automatically serves the updated data.
Token is stored in sessionStorage only. Never shared. Generate token ↗

How to Set Up

  1. Go to GitHub Settings → Tokens and generate a Classic token with repo scope.
  2. Enter the token, your GitHub username, repository name, and branch above.
  3. Make sure data/products.json exists in your repository.
  4. In your index.html, the product section auto-loads from this JSON file.
  5. Click Publish to GitHub any time to push your changes live.
